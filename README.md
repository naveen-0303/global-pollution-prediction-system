# Global Pollution Prediction System 🌍

AI-powered pollution prediction web application built using machine learning and environmental data.

## Features
- Pollution risk prediction using ML model
- Real-time AQI monitoring using WAQI API
- Interactive data visualization (charts & maps)
- Global pollution mapping
- 7-day pollution risk forecasting
- PDF report generation
- City pollution lookup with geospatial visualization

## Technologies
Python, Streamlit, Pandas, NumPy, Matplotlib, PyDeck, Machine Learning, REST API
